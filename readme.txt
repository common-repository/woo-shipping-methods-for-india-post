== Description ==

WooCommerce Shipping method for Indian Post is an Wordpress plugin that provides the following WooCommerce Shipping methods and Options

    1. Speed Post
    2. Registered Parcel
    3. Registered Parcel w/ VPP a.k.a COD

== Installation ==

1. Upload `woocommerce-shipping-method-for-indian-post` folder to the `/wp-content/plugins/` directory .
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Settings can be configured in the 'Shipping' tab of WooCommerce.
